# PaperHub.com

A web platform where students can upload previous year question papers and earn rewards.

## ⚙️ Features
- Upload question papers
- Reward point system
- Ad monetization
- Mobile-responsive layout

## 🔧 Tech Stack
- HTML, CSS, JavaScript
- PHP
- Google Ads integration
